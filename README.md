## [Module 00: The Basics of Mobile Development](./subjects/42_mobile_piscine_00.pdf)

### Exercise 00: A Basic Display

- Create a new project (I did it with flutter)
- Display centered text and a button
- Implement button press detection (log to console)
- Ensure responsiveness across devices

### Exercise 01: Say Hello to the World

- Build upon Exercise 00
- Toggle text between initial message and "Hello World!" on button press
- Introduce basic state management

### Exercise 02: Moar Buttons

- Create calculator UI layout
- Add AppBar with "Calculator" title
- Implement two text fields (expression and result)
- Add number buttons (0-9), decimal point, and operation buttons
- Include "AC" (reset), "C" (delete last character), and "=" buttons
- Log button presses to debug console
- Ensure responsive layout for various devices

### Exercise 03: It's Alive

- Implement calculator functionality
- Handle basic arithmetic operations (add, subtract, multiply, divide)
- Support multiple operations in one expression
- Allow negative and decimal number inputs
- Implement "C" and "AC" button functionality
- Handle potential errors (e.g., division by zero, invalid expressions)
- Ensure the app never crashes under any circumstances
